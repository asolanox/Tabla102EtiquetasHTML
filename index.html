Número Nombre de la Etiqueta Descripción Ejemplo de Uso Elemento Adicional
1 <html> Define el inicio de un documento HTML. <html></html> Contiene todos los elementos de la página.
2 <head> Contiene metadatos sobre el documento. <head></head> Incluye enlaces a CSS y scripts.
3 <title> Define el título del documento. <title>Mi Página</title> Se muestra en la pestaña del navegador.
4 <body> Contiene el contenido visible de la página. <body></body> Todo lo que se muestra al usuario.
5 <h1> Define un encabezado de nivel 1. <h1>Título Principal</h1> Usado para el título más importante.
6 <h2> Define un encabezado de nivel 2. <h2>Subtítulo</h2> Usado para subtítulos importantes.
7 <h3> Define un encabezado de nivel 3. <h3>Encabezado Secundario</h3> Usado para secciones dentro de un subtítulo.
8 <h4> Define un encabezado de nivel 4. <h4>Encabezado Terciario</h4> Usado para subsecciones.
9 <h5> Define un encabezado de nivel 5. <h5>Encabezado Cuaternario</h5> Usado para niveles más bajos de encabezado.
10 <h6> Define un encabezado de nivel 6. <h6>Encabezado Quinario</h6> El nivel más bajo de encabezado.
11 <p> Define un párrafo. <p>Este es un párrafo.</p> Usado para agrupar texto.
12 <br> Inserta un salto de línea. Texto antes del salto<br>Texto después del salto No tiene etiqueta de cierre.
13 <hr> Inserta una línea horizontal. <hr> Usado para separar contenido.
14 <a> Define un hipervínculo. <a href="https://www.ejemplo.com">Visitar Ejemplo</a> Usado para enlazar a otras páginas.
15 <img> Inserta una imagen. <img src="imagen.jpg" alt="Descripción"> No tiene etiqueta de cierre.
16 <div> Define una sección en un documento. <div>Contenido aquí</div> Usado para agrupar elementos.
17 <span> Define una sección en línea. <span>Texto en línea</span> Usado para aplicar estilos a partes del texto.
18 <strong> Define texto importante. <strong>Texto importante</strong> Generalmente se muestra en negrita.
19 <em> Define texto enfatizado. <em>Texto enfatizado</em> Generalmente se muestra en cursiva.
20 <small> Define texto de menor tamaño. <small>Texto pequeño</small> Usado para notas al pie.
21 <mark> Define texto resaltado. <mark>Texto resaltado</mark> Usado para destacar información.
22 <del> Define texto que ha sido eliminado. <del>Texto eliminado</del> Generalmente se muestra tachado.
23 <ins> Define texto que ha sido insertado. <ins>Texto insertado</ins> Generalmente se muestra subrayado.
24 <sub> Define texto subíndice. <sub>Texto subíndice</sub> Usado en fórmulas químicas.
25 <sup> Define texto superíndice. <sup>Texto superíndice</sup> Usado en exponentes.
26 <blockquote> Define una cita en bloque. <blockquote>Cita aquí</blockquote> Usado para citas largas.
27 <code> Define un fragmento de código. <code>Código aquí</code> Usado para mostrar código fuente.
28 <pre> Define texto preformateado. <pre>Texto preformateado</pre> Conserva espacios y saltos de línea.
29 <ul> Define una lista desordenada. <ul><li>Elemento 1</li></ul> Usa viñetas para los elementos.
30 <ol> Define una lista ordenada. <ol><li>Elemento 1</li></ol> Usa números para los elementos.
31 <li> Define un elemento de lista. <li>Elemento de lista</li> Usado dentro de <ul> o <ol>.
32 <table> Define una tabla. <table></table> Contiene filas y columnas.
33 <tr> Define una fila en una tabla. <tr><td>Celda 1</td><td>Celda 2</td></tr> Se utiliza dentro de una tabla.
34 <td> Define una celda en una tabla. <td>Contenido</td> Puede contener texto, imágenes, etc.
35 <th> Define una celda de encabezado en una tabla. <th>Encabezado</th> Por defecto, el texto es negrita y centrado.
36 <thead> Define el encabezado de una tabla. <thead><tr><th>Encabezado</th></tr></thead> Se utiliza para agrupar el encabezado de la tabla.
37 <tbody> Define el cuerpo de una tabla. <tbody><tr><td>Contenido</td></tr></tbody> Contiene las filas de datos de la tabla.
38 <tfoot> Define el pie de una tabla. <tfoot><tr><td>Total</td></tr></tfoot> Se utiliza para agrupar el pie de la tabla.
39 <caption> Define un título para una tabla. <caption>Título de la tabla</caption> Se coloca antes de <thead>.
40 <form> Define un formulario para la entrada de datos. <form action="submit.php"></form> Puede contener varios elementos de entrada.
41 <input> Define un campo de entrada. <input type="text" name="nombre"> Existen diferentes tipos de entrada (text, password, etc.).
42 <textarea> Define un área de texto para entrada de múltiples líneas. <textarea>Texto aquí</textarea> Se puede especificar el número de filas y columnas.
43 <button> Define un botón que puede ser presionado. <button>Enviar</button> Puede ser de tipo submit o button.
44 <select> Define un menú desplegable. <select><option>Opción 1</option></select> Puede contener múltiples opciones.
45 <option> Define una opción en un menú desplegable. <option value="1">Opción 1</option> Se utiliza dentro de un <select>.
46 <label> Define una etiqueta para un elemento de formulario. <label for="nombre">Nombre:</label> Mejora la accesibilidad del formulario.
47 <fieldset> Define un grupo de elementos relacionados en un formulario. <fieldset><legend>Información Personal</legend></fieldset> Se utiliza para agrupar elementos de formulario.
48 <legend> Define un título para un <fieldset>. <legend>Título del grupo</legend> Se coloca dentro de un <fieldset>.
49 <datalist> Define una lista de opciones predefinidas para un campo de entrada. <input list="browsers"><datalist id="browsers"><option value="Chrome"></datalist> Mejora la experiencia del usuario al autocompletar.
50 <output> Define el resultado de un cálculo. <output>Resultado</output> Se utiliza en combinación con formularios.
51 <progress> Define el progreso de una tarea. <progress value="50" max="100"></progress> Visualiza el progreso de una operación.
52 <meter> Define una medida escalar dentro de un rango conocido. <meter value="0.6" min="0" max="1"></meter> Se utiliza para mostrar datos cuantitativos.
53 <details> Define un control que el usuario puede abrir y cerrar. <details><summary>Más información</summary>Contenido adicional</details> Útil para mostrar información opcional.
54 <summary> Define un resumen para un elemento <details>. <details><summary>Más información</summary>Contenido adicional</details> Usado para mostrar u ocultar contenido.
55 <dialog> Define un cuadro de diálogo o ventana modal. <dialog>Este es un diálogo</dialog> Puede ser abierto o cerrado mediante JavaScript.
56 <canvas> Usado para dibujar gráficos mediante JavaScript. <canvas id="miCanvas"></canvas> Se puede manipular con el contexto 2D o WebGL.
57 <svg> Define gráficos vectoriales escalables. <svg width="100" height="100"><circle cx="50" cy="50" r="40" fill="red"/></svg> Ideal para gráficos que necesitan ser escalados.
58 <audio> Define contenido de audio. <audio controls><source src="audio.mp3" type="audio/mpeg"></audio> Soporta múltiples formatos de audio.
59 <video> Define contenido de video. <video width="320" height="240" controls><source src="video.mp4" type="video/mp4"></video> Permite la reproducción de video en la web.
60 <source> Especifica múltiples fuentes para elementos multimedia. <video><source src="video.mp4" type="video/mp4"></video> Optimiza la carga de medios según el navegador.
61 <track> Proporciona pistas de texto para elementos de video y audio. <video><track src="subtitulos.vtt" kind="subtitles"></video> Mejora la accesibilidad del contenido multimedia.
62 <iframe> Define un marco en línea para incrustar otro documento. <iframe src="https://www.ejemplo.com"></iframe> Permite mostrar contenido de otras páginas web.
63 <embed> Incrusta contenido externo, como multimedia. <embed src="archivo.pdf" type="application/pdf"></embed> Usado para mostrar documentos y otros tipos de contenido.
64 <object> Define un objeto que puede ser un documento o un multimedia. <object data="video.mp4" type="video/mp4"></object> Puede contener imágenes, videos, y más.
65 <param> Define parámetros para un objeto. <object><param name="autoplay" value="true"></object> Usado dentro de <object> para especificar configuraciones.
66 <map> Define un mapa de imagen. <map name="mapa"><area shape="rect" coords="34,44,270,350" href="ejemplo.html"></map> Permite crear áreas clicables en imágenes.
67 <area> Define un área dentro de un mapa de imagen. <area shape="circle" coords="100,100,50" href="ejemplo.html"> Usado junto con <map> para crear enlaces en imágenes.
68 <link> Define la relación entre el documento actual y un recurso externo. <link rel="stylesheet" href="estilos.css"> Usado para enlazar hojas de estilo CSS.
69 <meta> Define metadatos sobre el documento HTML. <meta charset="UTF-8"> Proporciona información sobre el documento.
70 <style> Define estilos CSS para el documento. <style>body { background-color: lightblue; }</style> Se puede usar para aplicar estilos directamente en el HTML.
71 <script> Define un script de programación. <script>alert('Hola Mundo');</script> Usado para incluir JavaScript en la página.
72 <noscript> Define un contenido alternativo para navegadores que no soportan <script src="script.js"></script> define un contenido alternativo que se mostrará a los usuarios que hayan deshabilitado los scripts
73 <base> Define la URL base para todos los enlaces relativos en un documento. <base href="https://www.ejemplo.com"> Se debe incluir dentro de la etiqueta <head>.
74 <address> Proporciona información de contacto para el autor o propietario de un documento. <address>Juan Pérez, Calle Falsa 123</address> Puede incluir un enlace a un correo electrónico.
75 <article> Define contenido independiente que puede ser distribuido o reutilizado. <article><h2>Título</h2><p>Contenido del artículo.</p></article> Ideal para blogs o publicaciones.
76 <aside> Define contenido relacionado que se puede considerar separado del contenido principal. <aside>Notas laterales</aside> Usado comúnmente para barras laterales.
77 <footer> Define el pie de página de un documento o sección. <footer><p>Derechos reservados</p></footer> Puede incluir enlaces a políticas de privacidad.
78 <header> Define el encabezado de un documento o sección. <header><h1>Título</h1></header> Puede incluir un logotipo o menú de navegación.
79 <main> Define el contenido principal de un documento. <main><p>Contenido principal aquí.</p></main> Solo debe haber un elemento <main> por página.
80 <nav> Define un conjunto de enlaces de navegación. <nav><a href="#home">Inicio</a></nav> Usado para menús de navegación.
81 <section> Define una sección en un documento. <section><h2>Título de sección</h2><p>Contenido.</p></section> Puede contener múltiples elementos relacionados.
82 <time> Representa un momento específico en el tiempo o un intervalo de tiempo. <time datetime="2024-09-11">11 de septiembre de 2024</time> Se puede usar para marcar fechas y horas, facilitando su interpretación por máquinas.
83 <progress> Representa el progreso de una tarea en curso. <progress value="70" max="100">70%</progress> Se puede usar para mostrar el progreso de descargas o tareas.
84 <template> Define un contenido que no se renderiza al cargar la página, pero puede ser utilizado posteriormente. <template><p>Este es un contenido de plantilla.</p></template> Útil para crear fragmentos de HTML que se pueden clonar y usar dinámicamente.
85 <picture> Proporciona una forma de especificar múltiples fuentes de imagen para diferentes condiciones. <picture>
  <source srcset="imagen.webp" type="image/webp">
  <img src="imagen.jpg" alt="Descripción de la imagen">
</picture> 		Permite la carga de imágenes optimizadas según el dispositivo o la resolución.
86 <source> Especifica múltiples fuentes para elementos multimedia como audio y video. <audio controls>
  <source src="audio.mp3" type="audio/mpeg"></audio>		 Se utiliza dentro de elementos como <audio> y <video> para mejorar la compatibilidad.
87 <track> Proporciona pistas de texto para elementos de video, como subtítulos. <video controls>
  <track src="subtitulos.vtt" kind="subtitles"></video>		 Mejora la accesibilidad de los videos al ofrecer subtítulos o descripciones.
88 <bdi> Aísla una parte del texto que puede ser afectada por el contexto de dirección del texto. <bdi>Texto en dirección opuesta</bdi> Útil en textos multilingües para evitar confusiones en la dirección del texto.
89 <bdo> Cambia la dirección del texto en el que se encuentra. <bdo dir="rtl">Texto de derecha a izquierda</bdo> Permite controlar la dirección del texto independientemente del contexto.
90 <ruby> Define una anotación de ruby, que es un texto adicional que se muestra sobre o junto a un texto. <ruby>漢字<rt>kanji</rt></ruby> Comúnmente usado en idiomas como el japonés para ayudar con la pronunciación.
91 <rt> Define el texto de ruby que se muestra junto a un elemento <ruby>. <ruby>漢字<rt>kanji</rt></ruby> Se utiliza para proporcionar la pronunciación o traducción de caracteres complejos.
92 <rp> Proporciona paréntesis alternativos para navegadores que no soportan ruby. <ruby>漢字<rp>(</rp><rt>kanji</rt><rp>)</rp></ruby> Mejora la accesibilidad en navegadores que no interpretan correctamente el ruby.
93 <b> Define texto en negrita, sin un significado especial. <b>Texto en negrita</b> Se utiliza para resaltar texto, aunque no implica énfasis.
94 <i> Define texto en cursiva, sin un significado especial. <i>Texto en cursiva</i> Se utiliza para resaltar texto, aunque no implica énfasis.
95 <u> Define texto subrayado. <u>Texto subrayado</u> Se utiliza para enfatizar texto, aunque puede ser confuso con enlaces.
96 <s> Define texto tachado. <s>Texto tachado</s> Indica que el texto ya no es relevante o correcto.
97 <q> Define una cita corta en línea. <q>Esto es una cita.</q> Los navegadores suelen agregar comillas automáticamente alrededor del texto.
98 <cite> Define el título de una obra, como un libro o una película. <cite>El Quijote</cite> Se utiliza para referirse a obras literarias o artísticas.
99 <abbr> Define una abreviatura o acrónimo. <abbr title="HyperText Markup Language">HTML</abbr> El atributo title proporciona la expansión de la abreviatura.
100 <data> Asocia un valor legible por máquina con un contenido. <data value="2024-09-11">11 de septiembre de 2024</data> Útil para marcar datos que pueden ser procesados por scripts.
101 <kbd> Define texto que representa la entrada del teclado. <kbd>Ctrl + C</kbd> Se utiliza para mostrar combinaciones de teclas o comandos.
102 <var> Define una variable en un contexto matemático o de programación. <var>x</var> es una variable en la ecuación. Se utiliza en contextos donde se necesita representar variables.
